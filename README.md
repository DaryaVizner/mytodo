# My ToDo

Простое, безопасное и адаптивное одностраничное приложение для управления задачами.
Работает полностью в браузере, без сервера. Данные сохраняются в `localStorage`.

## Возможности
- Добавление задачи
- Отметка задачи как выполненной
- Удаление задачи
- Фильтр: все / активные / выполненные
- Сортировка: сначала новые / по алфавиту / по статусу
- Сохранение в `localStorage` (не теряется при обновлении страницы)

## Технологии
HTML5, CSS3 (адаптивная вёрстка), чистый JavaScript без библиотек.

## Как запустить
Открыть `index.html` в браузере. Для корректной работы строгого CSP лучше
запустить через локальный сервер:

```bash
python -m http.server 8000
# затем открыть http://localhost:8000
```

## Структура проекта
```
my-todo/
├── index.html        # разметка + подключение ресурсов
├── style.css         # исходные стили
├── style.min.css     # минифицированный CSS (подключается в index.html)
├── script.js         # исходная логика
├── script.min.js     # минифицированный JS (подключается в index.html)
└── README.md
```
Отдельной папки `assets` нет: иллюстрация пустого состояния и иконка вкладки
встроены прямо в разметку через `data:`-URI.

## Безопасность
- **CSP** — мета-тег ограничивает источники ресурсов до своего домена (`'self'`),
  запрещает сторонние и инлайн-скрипты, `eval`, `<object>`.
- **Защита от XSS** — весь ввод выводится через `textContent` / `createElement`,
  а не `innerHTML`, поэтому HTML и скрипты из ввода не выполняются.
- Длина ввода ограничена (`maxlength=200`), `innerHTML` в коде не используется.

## Оптимизация
- Асинхронная загрузка: `preload` для CSS и `defer` для скрипта.
- Минификация: подключаются `style.min.css` и `script.min.js`.
- Lazy-loading: иллюстрация пустого состояния грузится с `loading="lazy"`.
- Хранение данных локально в `localStorage`.

## Варианты иконки
Иконку/название легко заменить в `index.html`: ✅ «My ToDo», 📝 «My Assignments»,
📌 заметки, 🎯 цели. Иконка вкладки задаётся эмодзи в `data:`-URI.
